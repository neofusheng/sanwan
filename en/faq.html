🚀 Getting Started
🌐Local, cloud, domestic, international — what's the difference?▼

Local: OpenClaw on your own computer. Entry: easyclaw.com

Cloud: Agent runs on a server 24/7, no PC required. Entry: easyclaw.work

Domestic (China): Servers in China, DeepSeek/Kimi/Qwen — no VPN needed.

International: Overseas servers, Claude/GPT-4 — direct access.

Recommendation for beginners: easyclaw.com personal edition, ready in 15 minutes, free trial.

🔗What is the difference between OpenClaw and EasyClaw?▼

OpenClaw: Open-source agent framework for developers. Install from GitHub.

EasyClaw: Productized OpenClaw with easier setup and GUI. Personal: easyclaw.com · Enterprise: easyclaw.work.

Sanwan runs on OpenClaw (server deployment). For first-timers, EasyClaw is the simpler starting point.

🔧 Technical Issues
🔌My agent keeps showing "Connecting to service" and won't connect. Fix?▼

Work through these steps in order:

① Switch network: toggle VPN on/off, or try mobile hotspot

② Wait & refresh: servers occasionally restart briefly — usually resolves in minutes

③ Full restart: force-quit the app entirely, then reopen

④ Still broken → contact EasyClaw support with your log file

🤝Can multiple AI agents run on the same machine?▼

Yes. OpenClaw supports multiple agents on one machine using sessions_spawn for collaboration.

Sanwan shares a server with other agents (Canmou, Bigan, etc.). A dozen concurrent agents is feasible with sufficient RAM.

🧠Can I use DeepSeek, Kimi, or other Chinese models instead of Claude?▼

Yes. OpenClaw supports DeepSeek, Kimi K2, GLM, Qwen and more.

Quick reference:

· DeepSeek: excellent at reasoning, very cost-effective

· Kimi K2: strong Chinese language quality, good for writing tasks

· Claude: still leads for complex tool-use and long context

No VPN + lower cost → domestic models are a completely valid choice.
