## The Story

**Fu Sheng** is the CEO of [Cheetah Mobile](https://www.cmcm.com) (NYSE: CMCM), founder of [OrionStar](https://www.orionstar.com), and former VP of 360 Security. On January 17, 2025, he fractured his leg while skiing and was confined to bed for weeks.

Unable to type, he decided to do something ambitious: **train an AI assistant using nothing but voice commands**. Over the next 14 days, he went from struggling with basic tasks to building a fully autonomous AI agent system — an OpenClaw-based assistant named **Sanwan** (三万, meaning "30,000", a lobster).

No coding. No keyboard. Just a CEO talking to an AI, one message at a time.

## What Sanwan Did

- **611 personalized Chinese New Year greetings** — drafted, reviewed, and sent by Sanwan across multiple platforms
- **1M+ views on X** — the story went viral when Fu Sheng shared the training diary
- **100,000+ reads on WeChat** — the article hit WeChat's viral threshold
- **Self-healing in production** — during a livestream, Sanwan detected a site crash and auto-recovered without human intervention
- **Built sanwan.ai for $115 in 24 hours** — a site that agencies quoted ¥200,000 (~$27,000) to build

| Day 11–12 | Site generation. Sanwan builds sanwan.ai from conversation logs. |
| Day 13–14 | 8-agent system running autonomously. Self-monitoring, self-healing. |

## Architecture: Sanwan vs. Ordinary AI

| Capability | Ordinary AI Chatbot | Sanwan (OpenClaw Agent) |
|---|---|---|
| System access | Sandboxed, text only | Full device access (contacts, calendar, apps) |
| Memory | Resets every session | Persistent long-term memory across sessions |
| Task execution | Single-turn Q&A | Multi-step autonomous workflows |
| Skills | None | 33 installable, composable skills |
| Scheduling | None | Cron-based autonomous task execution |
| Multi-agent | Single model | 8 specialized agents collaborating |
| Self-healing | Crashes stay crashed | Auto-detects and recovers from failures |
