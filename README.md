# 🦞 三万 · 龙虾养成日记

**sanwan.ai** — 一个CEO骨折在家14天，从零养出一只AI龙虾的故事。

## 页面
- **首页** — 故事入口 + 4篇精华日记
- **日记** — 14天完整养成日记（11篇）
- **科普** — AI Agent是什么？龙虾和普通AI有啥区别？
- **技能** — 三万同款技能商城
- **EasyClaw** — 养你自己的AI龙虾

## 技术栈
- 纯HTML/CSS/JS，零依赖
- PPT白板手绘风格
- 移动端优先

## 数据
- 1157条消息，22万字对话
- 8个AI Agent协作
- 33个可安装技能
