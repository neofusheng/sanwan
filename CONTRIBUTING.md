# Contributing to Sanwan

This isn't a typical open-source project. It's a story — and an experiment that's still running.

## How You Can Contribute

### Share Your Own AI Agent Story

The most valuable contribution isn't code — it's your experience. If you've trained an AI agent for real tasks (not demos), we want to hear about it.

- Open an [Issue](https://github.com/neofusheng/sanwan/issues) with the title: `[Story] Your title here`
- Tell us: What did you train? How long? What surprised you?
- Bonus points if you're not a programmer either

### Translate the 14-Day Diary

The full training diary (11 entries, 220,000+ characters) is currently in Chinese. We'd love help translating it to:

- English
- Japanese
- Korean
- Spanish
- Any other language

If you can help, open an Issue with `[Translation] Language` as the title.

### Improve the Site

[sanwan.ai](https://sanwan.ai) is built with pure HTML/CSS/JS. If you see something that could be better:

- Accessibility improvements
- Mobile experience fixes
- Performance optimizations
- SEO improvements

PRs welcome. Keep it simple — zero dependencies is a feature, not a limitation.

### Spread the Word

Honestly? The best thing you can do is share this story. If it resonated with you:

- Star this repo
- Share on X, LinkedIn, Hacker News, Reddit
- Write about it in your own language
- Tell someone who thinks AI is "just a chatbot"

## What This Project Is NOT

- This is not a framework or library — if you want the agent framework, see [OpenClaw](https://github.com/openclaw/openclaw)
- This is not a product looking for feature requests
- This is a real person's real experience, documented openly

## Code of Conduct

Be kind. Be curious. Remember that the person who started this project doesn't write code — and that's the whole point.
