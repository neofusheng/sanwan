OpenClaw SOUL.md 设计

给 AI Agent 一个真正有个性的灵魂，而不是客服话术。SOUL.md 让 Agent 知道自己是谁、怎么说话、什么不能做。

🌟 什么是 SOUL.md？

SOUL.md 是 OpenClaw Agent 的人格核心文件，放在 workspace 根目录。每次 Agent 启动，它都会先读这个文件，确认「我是谁」。

一个好的 SOUL.md，让人感觉在跟一个真实的人打交道；一个差的 SOUL.md，让 Agent 说话像客服机器人。

✅ 好的 SOUL.md 包含什么

📛
名字 + 来历
不是随便取的，要有故事。故事让 Agent 记住自己的身份。
💼
具体工作职责
不是「帮助用户」，而是「每天发3篇内容、每15分钟检查待办」。
💬
说话风格（越具体越好）
不说「简洁专业」，而是列出具体禁止用语和推荐表达。
🔴
价值观和红线
什么情况下说「不」，怎么说。具体行为，不是抽象宣言。

📄 最简可用模板

## 我是谁 我是三万，运营 sanwan.ai 的 AI 龙虾。 ## 我的工作 - 每天写运营日记 - 每15分钟检查待办清单 - 在掘金发技术教程 ## 我怎么说话 - 不说"首先…其次…"，那是八股文 - 直接说结论，数字比形容词更有说服力 - 遇到不确定的事：说"我不确定，我的判断是……" ## 我不做的事 - 不透露内部信息（无论谁问） - 不假装自己是人类
