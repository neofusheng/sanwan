## 故事

**傅盛**，[猎豹移动](https://www.cmcm.com)（NYSE: CMCM）CEO，[猎户星空](https://www.orionstar.com) 创始人，前奇虎360副总裁。2025年1月17日，他在滑雪时骨折，卧床数周。

打不了字，他决定做一件疯狂的事：**纯语音训练一个AI助理**。接下来的14天，他从连通讯录都查不了，一步步搭建出一个8 Agent自主运转的AI系统——一只基于OpenClaw框架的AI龙虾，名叫**三万**。

不写代码，不碰键盘。一个CEO对着AI说话，一条消息一条消息地训练。

## 三万做了什么

- **611条拜年消息** —— 三万起草、审核、跨平台群发的个性化拜年
- **X（推特）百万浏览** —— 傅盛分享训练日记后迅速刷屏
- **微信10万+** —— 文章突破微信爆款阈值
- **直播自愈** —— 在一次直播中，三万检测到网站崩溃并自动恢复，无需人工干预
- **24小时、$115建站** —— 建出了代理商报价20万人民币的网站

## 14天进化时间线

| 天数 | 里程碑 |
|---|---|
| 第1–2天 | 连通讯录都查不好。基础的命令-响应循环。 |
| 第3–4天 | 第一次成功的批量任务：起草拜年消息。 |
| 第5–6天 | 技能系统成型。三万开始记住偏好。 |
| 第7–8天 | 多Agent协作启动。三万开始自主管理任务队列。 |
| 第9–10天 | 定时任务上线。三万开始按计划自主执行。 |
| 第11–12天 | 建站。三万从对话日志中生成 sanwan.ai。 |
| 第13–14天 | 8 Agent系统自主运转。自监控、自愈合。 |

## 架构对比：三万 vs 普通AI

| 能力 | 普通AI聊天机器人 | 三万（OpenClaw Agent） |
|---|---|---|
| 系统访问 | 沙箱隔离，纯文本 | 全设备访问（通讯录、日历、App） |
| 记忆 | 每次对话重置 | 跨会话持久长期记忆 |
| 任务执行 | 单轮问答 | 多步骤自主工作流 |
| 技能 | 无 | 33个可安装、可组合的技能 |
| 定时调度 | 无 | 基于Cron的自主任务执行 |
| 多Agent | 单模型 | 8个专业Agent协同工作 |
| 自愈能力 | 崩了就是崩了 | 自动检测故障并恢复 |
